=== MEDIEVAL SOP (Standard Operating Procedure) ===

This is EXACTLY how medieval apothecaries wrote recipes!

They would NOT write the full procedure 33 times. Instead:

1. Define the BASE PROCEDURE once (the repeating block)
2. List the VARIATIONS/PARAMETERS (the initiators)
3. Each [*] means "Apply the base procedure with this parameter"

MODERN ANALOGY:
```python

    """Base Procedure A"""
```
